
## Серверная часть 

1. Перейдите в папку server:
   cd server
2. создайте .env и поместите туда :
   SERVER_PORT=3000
3. Установите все необходимые зависимости:
   npm i
4. Запустите сервер:
   npm run start

## Клиентская часть

1. Перейдите в папку client:
   cd client
2. Создайте .env и поместите туда:
   VITE_API_URL=http://localhost:3000/contact
3. Установите все необходимые зависимости:
   npm i
4. Запустите клиентскую часть:
   npm run dev
